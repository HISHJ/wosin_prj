## :pushpin: 주요기능


### 로그인
- DB값 검증
- ID찾기,PW찾기
- 로그인 했을때 session생성

### 회원가입
- DB값 추가
- 아이디,비밀번호,회원정보 암호화
- 주소API사용
- 아이디 중복검사
- 아이디 비연결성 : 동시에 동일한 아이디가 입력되어도 한개만 DB에 추가됨

- 회원정보 복호화
- PW변경,회원정보수정,회원탈퇴
- 주소API사용
